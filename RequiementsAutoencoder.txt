Для просмотра внешней оболочки требуется компиляция файлов app.py Autoencoder.ipynb index.html style.css script.js в одну папку.
Для запуска модели при просмотре внешней оболочки нужно выгрузить графический файл клавишей 'Загрузить изображение' для обработки с использованием модели нейронной сети. После загрузки файла на сайт нужно нажать клавишу 'Преобразовать изображение' для использования готовой модели Autoencoder.ipynb
